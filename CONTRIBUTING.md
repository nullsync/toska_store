# Contributing to ToskaStore

Thank you for your interest in ToskaStore.

## Contribution Policy

This project is developed and maintained by Abstractive Machines LLC. **External contributions are not accepted at this time.** All code in this repository is authored by Abstractive Machines LLC to maintain clear intellectual property ownership.

If you have found a bug or have a feature suggestion, please open an issue. We appreciate feedback and will consider it for future development.

## For Internal Development

If you are a contributor authorized by Abstractive Machines LLC:

1. Follow Elixir coding conventions and the existing code style
2. Write tests for new features
3. Run `mix format` before committing
4. Keep commits focused and write clear commit messages

## Questions

For questions about this policy or the project, open an issue or contact the maintainer [@nullsync](https://github.com/nullsync).
